\# HeyGen Video Generator



Bu proje HeyGen API kullanarak talking photo videoları oluşturur.



\## Kurulum



1\. Repoyu klonlayın

2\. `.env` dosyasına kendi API anahtarınızı ve diğer bilgileri girin

3\. Kütüphaneleri yükleyin:



pip install -r requirements.txt



4\. Çalıştırın:



python main.pys
